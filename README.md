# orders-api

Projeto backend desenvolvido com Spring Boot durante estudos de Java e desenvolvimento de APIs REST.

## Objetivos do projeto

- Estruturação de API REST
- Organização em camadas
- Spring Boot
- JPA / Hibernate
- Banco de dados H2
- CRUD
- Relacionamentos entre entidades
- Enum
- Tratamento de exceções

## Tecnologias utilizadas

- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- Maven
- H2 Database
- PostgreSQL
- Postman

## Status do projeto

Projeto em desenvolvimento e evolução contínua durante estudos e práticas de backend Java.
